TYDZIEN III

Jeszcze trochę o stylowaniu

  • Styl liniowy ( inline) - Styl przy znaczniku z wykorzystaniem atrybutu style - nie ma się co rozpisywać, po prostu nie używamy

    <znacznik style=”color:yellow;”> treść</ znacznik>

  • Stylowanie klasami

    <znacznik class=”name”> treść</ znacznik>

    Aby wystylować taką klasę używamy selektora klasy: .name

  • Stylowanie za pomocą id

    <znacznik id=”name”> treść</ znacznik>

    Aby wystylować taką klasę używamy selektora id: #name

  • Jest jeszcze coś takiego jak !important którego nie powinniśmy używać ponieważ nadpisuje wszystkie inne rzeczy które są zdefiniowane.

Specyficzność w CSS 🤪

  • Jest coś takiego jak specyficzność w CSS i najlepiej podejść do tego metodą medalową :)

    | !important | style="" | #name | .name | element |

  • Jeśli specyficzność jest równa - pierwszeństwo ma selektor "bliżej" elementu
  • Dodajemy klasy do wszystkich elementów i unikamy zagnieżdżania co rozwiąże problem z liczeniem specyficzności.
  • Nie zaczynaj od cyfry.
  • Staraj się, żeby element miał jedną klasę (nie zwiększaj specyficzności bez potrzeby).

Nazewnictwo klas

  • Używaj pojedynczych słów z j. angielskiego a jeśli potrzeba więcej robimy to poprzez kebab-case.
  • Nazywaj rzeczy zgodnie z ich funkcją na stronie.
  • Startuj z małej litery np. header nie Header.
  • Nie zaczynaj od cyfry.
  • Staraj się, żeby element miał jedną klasę (nie zwiększaj specyficzności bez potrzeby).

TYDZIEN II

Podstawy CSS

  • CSS =Cascading Style Sheet - odpowiada za prezentację, wpływa na wygląd elementów HTML na ekranie, wydrukach itp.
  • Przeglądarki potrafią różnie interpretować CSS.
  • Stylowanie HTML odbywa się w następujący:
    • Wpisując reguły CSS w z znacznik <style> umieszczony w sekcji <head>
    • Dodanie zewnętrznego pliku stylów umieszczonego w <head> za pomocą znacznika <link>

      <link href="style.css" rel="stylesheet" type="text/css">

  • Budowa reguły CSS:

    selektor{

    właściwość:

    wartość;

    }
