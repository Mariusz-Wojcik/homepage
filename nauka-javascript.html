TYDZIEN III

JavaScript != Java

  • Będziemy uczyć się ECMAScript 2015+(ES6+)
    • ECMAScript jest to ustandaryzowany przez organizację ECMA skryptowy język programowania.
    • JavaScript implementuje rzeczy ze specyfikacji ECMAScript.
    • Rozwój ECMAScript mocno przyspieszył przez szerokie wykorzystywanie JavaScriptu.
    • ES6 (i nowsze) wprowadzają wiele udogodnień dla developerów.
    • Nowe feature'y mogą nie być wspierane we wszystkich przeglądarkach.
  • Z czego składa się JavaScript?
    • Wartości
    • Operacje porównywania
    • Zmienne
    • Literały
    • Funkcje
    • Obiekty

Poskryptujmy!

  • Znacznik <script> - dodajemy na samym dole przed zamknięciem body
    • <script>
       alert('Siema!')
       </script>
    • <script
       src="main.js" >
      </script>
  • Backtick / backquote ` - używamy kiedy zamieszczamy jednocześnie tekst i stałe:
    console.log(`Nazywam się ${ name } i mam ${ age } lat`);
  • Do nazywania stałych / zmiennych używamy camelCase.

Trochę praktyki JS

  • alert('Siema!'); - wyskakujące okienko
  • console.log('Siema!'); - zapis w konsoli devtools
  • stałe w JS (tekst): const name = 'Mariusz'
  • stałe w JS (liczby): const age = 18
